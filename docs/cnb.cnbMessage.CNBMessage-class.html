Class CNBMessage

source code

This is the message object. Every message sent through a room or directly, will be converted into this class so the modules can deal with standard format.

Type of the message. In xmpp, Should be one of these: chat | groupchat | normal (seems to be normal only from ejabberd servers) Type of the message. In irc, it should be one of these: pubmsg | privmsg | notice
