enProcessCmd

Determine if the module is processed from a command. If yes, the module is accessed by sending specific commands to bot (ex: .fact would reply a Chuck Norris fact).
Type:
bool

True

enProcessPattern

Determine if the module is processed from a pattern. If yes, the module is accessed by detecting some patterns on chat rooms or user behaviors (ex: 3x "k" message will output a "approuved")
Type:
bool
